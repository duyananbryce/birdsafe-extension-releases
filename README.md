# BirdSafe Releases

This repository is the public distribution channel for the BirdSafe browser extension.

Source code is not published here. This repository is used for:

- Release notes
- Downloadable extension build packages
- Installation instructions

## Install

1. Open the [Releases](https://github.com/duyananbryce/birdsafe-extension-releases/releases) page.
2. Download the latest `BirdSafe-extension-*.zip` asset.
3. Unzip the package on your computer.
4. Open your Chromium-based browser and go to `chrome://extensions`.
5. Enable `Developer mode`.
6. Click `Load unpacked`.
7. Select the extracted `dist` folder.
